Software Engineering

Studying software engineering I have learned a lot about designing, developing, and maintaining software. I made some management systems for universities and libraries, as well as other small projects like electricity bill calculators or adding a location to a website using the geocoding API. I learned that it is easier to develop an application than it is to maintain it and that sometimes users might do things I never thought of. However, these challenges are what make software engineering fun and exciting since you have to find the best solution for these problems.
